# 第10章 文件和异常
## 文件的读取
1. 直接读取整个文件内容到内存
```python
file_path = "./pi_digits.txt"
with open(file_path) as file_object:
    contents = file_object.read()
    print(content)
```
2. 逐行读取
>直接for循环open()得到的文件对象实例
```python
with open(file_path) as file_object:
    for line in file_object:
        print(line)
```
